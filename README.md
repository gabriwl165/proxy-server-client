## Compile protofiles

First of all, you must install dependecies in node application
```bash
sudo npm install
```

With this command you can just build the proto's files
```bash
npm run build-proto
```

With this command you can do everythin in one place.
-  Build the proto's
-  Transpile the TS code
-  Run the JS Server
```bash
npm run dev
```
